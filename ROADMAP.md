## ✅ v1.3.1 — Тұрақтылық жақсартулары (Ағымдағы)

- [x] Платонус JWT токені мерзімі бітсе **автоматты жаңарту** (_pc cookie арқылы)
- [x] Пән картасына басқанда **күн бойынша баллдар** көрінеді (L / Lab / SRSP)

- [x] `attendance.svelte` — `.value` → `.data` буг түзетілді (кесте шықпайтын)
- [x] `handle_api_error` — `TimeoutError` → 408 қайтарады (кэш сақталады)
- [x] Fetch timeout 10с → **20с** (univer.kstu.kz), Платонус → **15с**
- [x] `platonus.py` — 401/403 жауабы `None` ретінде өңделеді (refresh триггері)
- [x] `platonus.py` — барлық `except:` → `except Exception:` түзетілді
- [x] `platonus.py` — `_pt_headers_and_cookies()` хелпері (код қайталанбайды)
- [x] `platonus_login` — try/except қорғауы қосылды
